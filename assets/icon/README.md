# Datamosher icon assets

This folder contains the word-based Datamosher app icon.

Files:

- `datamosh-icon.svg` - source vector icon.
- `datamosh-icon-1024.png` - full-size PNG for README/GitHub/project pages.
- `datamosh-icon-512.png`, `datamosh-icon-256.png`, etc. - resized PNG exports.
- `datamosh.icns` - macOS app icon file for a future standalone `.app` build.

Design notes:

- Dark rounded-square macOS-style base.
- Large readable `DATA` / `MOSH` wordmark.
- Cyan/magenta chromatic offsets and horizontal glitch bars.
- Intended to stay simple and text-based while still feeling corrupted/video-native.
