# Security Policy

Datamosher is intended to run locally on your own machine.

## Credentials

This project should not contain credentials. Do not commit:

- API keys or tokens
- Apple Developer passwords or app-specific passwords
- Signing certificates or provisioning profiles
- `.env` files

## Local media

Uploads and generated videos are stored locally under `data/` by default and are ignored by git.

## Reporting issues

If you find a security issue, open a private report through GitHub Security Advisories if the repository has them enabled, or contact the maintainer directly.
