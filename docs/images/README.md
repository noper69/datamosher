# Screenshots and demo media

Place lightweight project images here before submitting to the hackathon.

Recommended files:

- `ui.png` - screenshot of the main Datamosher interface.
- `before-after.png` - comparison image from your demo footage.
- `demo.gif` - optional short optimized GIF, preferably under 5-10 MB.

Avoid committing large raw videos. Put full demo videos on YouTube, Vimeo, Loom, or a GitHub Release and link them from the README.
